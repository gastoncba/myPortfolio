Vegan Eat!

Vegan EAT! es una aplicación web que ofrece una amplia variedad de opciones de comidas veganas y vegetarianas. Los usuarios pueden explorar y seleccionar los platos que más les gusten y realizar pedidos personalizados, especificando detalles como la dirección de entrega, el método de pago y el horario deseado.
Este proyecto se encuentra actualmente no está alojado en ningún servidor. Sin embargo, el repositorio del código fuente está disponible para su acceso.
Repositorio de Vegan EAT

  • vue-google-autocomplete

Dessert Now

Se trata de una aplicación que permite a los usuarios comprar diferentes postres disponibles, organizados en tres categorías principales: Tortas, Helados y De Frutas. A través de la aplicación, los usuarios pueden explorar los postres disponibles en cada categoría, seleccionar y agregar al carrito de compras aquellos que deseen comprar, y realizar la compra de forma segura y sencilla. En resumen, esta herramienta ofrece una experiencia de compra en línea fácil y accesible para los amantes de los postres.

Desarrollo y Tecnológias utilizadas:

Para el desarrollo del frontend de esta aplicación, se optó por utilizar la potente librería React JS. Asimismo, para enfocarnos en la lógica de la aplicación, se incorporaron varias librerías adicionales, incluyendo Semantic UI, Material UI y Formik.
Cabe destacar que la aplicación se encuentra alojada en Firebase, lo que nos brinda una sólida base para su despliegue y manejo de datos en tiempo real.
  • Dessert Now - Repositorio
  • React
  • Semantic UI
  • Material UI
  • Formik
  • Firebase

Para la parte backend de la aplicación, se implementó una API REST desarrollada en Node.js, la cual se encarga de gestionar la comunicación entre el cliente y el servidor. Para la persistencia de los datos, se ha optado por el sistema de bases de datos NoSQL, MongoDB, el cual ha sido integrado con la API para asegurar un manejo eficiente y escalable de los datos de la aplicación
El backend se encuentra alojado en https://railway.app/.
  • API Dessert Now!
  • Railway
  • MongoDB

Fast Tour

Se trata de una aplicación simuladora que permite a los usuarios adquirir diversos paquetes turísticos. La aplicación brinda la posibilidad de buscar entre todos los viajes disponibles, filtrarlos según fechas y precios, y buscar lugares y países específicos para encontrar los viajes disponibles en dichas regiones. En resumen, esta herramienta ofrece una experiencia completa de búsqueda y compra de paquetes turísticos a medida para cada usuario.

Desarrollo y Tecnológias utilizadas:

La aplicación fue desarrollada utilizando React JS para el frontend. Se integraron las bibliotecas adicionales Material UI y Ant Design para agilizar el proceso de diseño y desarrollo de la interfaz de usuario.
La aplicación está alojada en DigitalOcean, lo que permite un acceso rápido y seguro desde cualquier lugar.
  • Fast Tour - Repositorio
  • React
  • Typescript
  • Material UI
  • Ant Desing

La parte backend de la aplicación se ha implementado utilizando una API REST desarrollada en Node.js y para la persistencia de los datos se ha utilizado el sistema de base de datos PostgreSQL. Adicionalmente, se ha utilizado TypeORM como ORM (Object Relational Mapping) para asegurar una correcta gestión de la base de datos y facilitar las operaciones de lectura y escritura de los datos. En resumen, se ha utilizado tecnología de última generación para garantizar un alto rendimiento y fiabilidad en el funcionamiento de la aplicación.
El backend se encuentra alojado en https://railway.app/.
  • API Fast Tour
  • Railway
  • TypeORM
  • PostgreSQL
